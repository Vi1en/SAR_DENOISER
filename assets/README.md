# Assets

## `images/`

Static and generated **figures** (PNG) that used to sit in the repository root: comparison plots, demo outputs, etc.

- **Dataset patches** stay under `data/` (e.g. `data/sample_sar/processed/`).
- **Evaluation run plots** stay under `results/` (per-run outputs).
- Scripts save new root-level-style plots here via `paths.ASSETS_IMAGES` in `paths.py`.
